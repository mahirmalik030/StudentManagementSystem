# Smart Student Management System
## Project Description
Smart Student Management System is a Java Swing based desktop application developed for the Software Construction and Development (SCD) Lab Project.
The system demonstrates:
* Event Handling
* Exception Handling
* Code Refactoring
* File Handling
* Singleton Design Pattern
* Git & GitHub Version Control
* Unit Testing using JUnit
## Features
* Add Student
* Save Student Records to File
* Calculate Student Grade
* Input Validation
* Custom Exception Handling
* Clear Form
* Graphical User Interface (GUI)
## Technologies Used
* Java
* Java Swing
* Eclipse IDE
* Git
* GitHub
* JUnit

## Setup Instructions
1. Install JDK 21
2. Install Eclipse IDE
3. Import Project
4. Run Main.java
5. Use the GUI to add students and calculate grades
